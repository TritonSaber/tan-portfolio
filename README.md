# tan-portfolio
### Justine Joshua Tan's portfolio - know who I am!

**Live Website**: https://justine-tan-portfolio.vercel.app/

# Description
This is a simple portfolio built to showcase my computer science skills, about me, my projects, and to get more of an understanding in developing a website. Website built from scratch without using a drag-and-drop website builder primarily using SvelteKit as the main framework, Tailwind CSS, Typescript, and Vercel as deployment.

# Current Project Setup
* **Framework:** [SvelteKit](https://svelte.dev/docs/kit/introduction)
* **CSS Framework:** [Tailwind CSS](https://tailwindcss.com/)
* **Back-end Language:** [Typescript](https://www.typescriptlang.org/)
* **Deployment:** [Vercel](https://vercel.com/)
* **Icons:** [Bootstrap Icons](https://icons.getbootstrap.com/) & [Heroicons](https://heroicons.com/)
* **Logo:** Made by me in [Canva](https://www.canva.com/)
* **Fonts:** [Inter](https://fonts.google.com/specimen/Intern) & [Google Sans Code](https://fonts.google.com/specimen/Google+Sans+Code)
